# Car Simulator

Python tooling workspace for CAN decoding, Traffic Simulation, and experiments.

## Quick Start

### 1. Prerequisites

- `uv` installed
- Virtual CAN setup (e.g., by running `../setupcan0.sh` to setup can0)

### 2. Setup

Navigate to the project and install dependencies:

```bash
cd IoT/car_simulator
uv sync
```

### 3. Usage

Run the simulator using `uv run`. For example:

```bash
uv run python main.py
```
